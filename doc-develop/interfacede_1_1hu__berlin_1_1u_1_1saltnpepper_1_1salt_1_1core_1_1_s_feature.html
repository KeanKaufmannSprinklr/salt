Detailed Description

An object of class SFeature is a label and can be added like every other label to a node, relation, graph, layer or to another label. Like every label a SFeature object consists of the main fields Label#getNamespace(), Label#getName() and Label#getValue().

A SFeature could be used, to add some information to any Salt object during a processing. So for instance you can store any state like 'already processed' or other non-linguistic and non meta annotations like 'having the color red' to it.
