Detailed Description

The salt project is a kind of a super container for a Salt model.

It contains a set of corpus structures, which could contain a set of document structures each. This makes the SaltProject object the entry point for a salt model. In case you have more than one salt project, you can name it by setName(String)

Map<Identifier, URI> de.hu_berlin.u.saltnpepper.salt.common.SaltProject.getDocumentGraphLocations ( )

This method extracts the location of all SDocumentGraph objects, which are persist in a file and are currently not loaded as model into main memory.

The returned value is a map, containing the SElementId as an identifier and the location of the SDocumentGraph object as an URI. To extract the locations, the method runs through all SDocument objects (being contained in the current SaltProject) containing the SDocumentGraph objects and checks if the SDocumentGraph object is loaded or peristed.

Returns
a map of SElementId of SDocument objects and the corresponding location as URI, an empty Map, if no SDocumentGraph object is persisted
